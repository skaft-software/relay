# Security Policy

## Supported Versions

Relay currently supports security reports for the latest release only:
- `v0.1.1`

## Reporting a Vulnerability

Please do not disclose vulnerabilities publicly before maintainer review.

If GitHub private security advisories are available for this repository, open a private advisory. Otherwise, contact the maintainer directly.

Relay handles API keys, auth headers, and proxy traffic. Reports involving credential leakage, authentication bypass, request logging exposure, and header forwarding behavior are treated as security-sensitive.
