# Contributing

Thanks for your interest in Relay.

Relay is not currently accepting public contributions.

At this stage, the project architecture and compatibility surface are still being stabilized. As a result, issues and pull requests from outside contributors may be closed without review.

Selected collaborators and team members may contribute by prior arrangement with the maintainer.

Unless invited otherwise, please use GitHub issues only for:
- Reproducible bug reports
- Compatibility reports
- Security-adjacent concerns

For security vulnerabilities, please see `SECURITY.md`.
